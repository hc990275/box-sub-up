# Box-sub-up

这是一个专为 **Box for Root** (Magisk/KernelSU/APatch 模块) 打造的订阅自动更新扩展工具。基于最新的 Clash API，提供极其简单、高效的无感全自动订阅更新体验。

## 🚀 核心功能 (v2.0.5)

- **无感适配**：完全抛弃繁琐的文件解析与路径映射，直接调用代理核心自带的 RESTful API 进行更新。
- **一键全量更新**：只需点击一次"一键更新"，即可自动遍历并刷新所有在线订阅项。
- **动态发现机制**：每次更新时自动探测所有的 Providers，无论你的配置怎么改，只需保证配置在 Box 中加载成功，更新器就能立刻发现。
- **流量/到期可视化**：在全新的 WebUI 面板中，可以直接查看每个订阅项目的流量使用情况、更新时间和过期天数。
- **完全兼容 Box**：自动跟随 `settings.ini` 探测核心变动，自动读取控制端口和密钥，开箱即用。

## 🛠️ 工作原理

1. **侦测阶段**：脚本启动或 WebUI 加载时，会解析 Box 系统的 `settings.ini`，确定活跃核心获取对应的 `external-controller` 端口和 `secret`。
2. **列表获取**：通过 `GET /providers/proxies` 请求代理核心，动态获取当所有的订阅项（Providers）名称。
3. **精准触发更新**：脚本自动遍历所有项目，调用 `PUT /providers/proxies/{name}` 触发核心自行去下载、更新并平滑重载该订阅的节点。
4. **反馈阶段**：将每一个项目的 HTTP 状态码记录至 `/sdcard/Android/sub.log`，并在面具描述中动态呈现更新状态。

## 📦 安装说明

1. 确保已安装 **Box for Root** 模块并正常运行（内核必须支持 API 特性，推荐最新版 mihomo 或 sing-box）
2. 在 Magisk/KernelSU/APatch 中刷入本模块的 zip 包
3. 重启设备
4. 在面具模块列表中点击本模块的"设置"图标进入最新的 WebUI 面板
5. 你可以查看订阅状态，一键手动更新，或设置后台自动守护周期（默认 30 分钟）
