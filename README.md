# Sing-box 订阅自动更新模块 (Magisk / KernelSU)

这是一个将 sing-box 订阅更新脚本模块化的项目。

## 功能特性
- **开机自启**：无需手动启动脚本。
- **现代化 WebUI**：在 Magisk / KernelSU APP 中直接修改刷新频率。
- **实时日志**：可视化监控更新任务执行状态。
- **手动触发**：Action 按钮支持即刻同步。

## 安装说明
1. 将本目录所有文件打包为 `.zip`。
2. 在 Magisk / KernelSU / APatch 管理器中选择从本地安装。
3. 重启设备即可生效。
